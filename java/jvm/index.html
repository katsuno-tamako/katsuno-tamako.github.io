JVM 自动内存管理

运行时数据区域

程序计数器 （Program Counter Register）

程序计数器是一个较小的内存空间，可以看作是当前线程所执行的字节码的行号指示器。 Java虚拟机的概念模型里，字节码解释器工作时就是通过改变这个计数器的值来选取下一条需要执行的字节码指令，他是程序控制流的指示器，分支、循环、跳转、异常处理、线程回复等基础功能都需要依赖这个计数器完成。
